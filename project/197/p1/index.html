Project 1

Type: Folding Booklet

This is a project where students will interview a partner using a set of questions, with the goal of fostering a sense of closeness and vulnerability. Students will then create a folding booklet that visualizes their partner’s unique personality and character using abstract pattern development, storytelling, poetry, quotes, lyrics, etc. Emphasis will be placed on original concept development and exploring diverse processes and techniques. The final mock-up will be presented in both physical and digital formats, with background images created using analog methods.
