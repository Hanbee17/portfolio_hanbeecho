Project 4

Type: Process Book

The objective of the process book is to provide a comprehensive view of the students' workflow and completed work during the semester. It should allow viewers to understand the creative process of the student and the progression of each project from the initial ideas to the final design. The deliverable for this project is a PDF file that includes a table of contents, a project outline, all notes, thumbnails, concepts, sketches, a 1-2 page overview of the final design, and a written analysis of each project, which includes what worked well, what did not work, and what the student learned.
